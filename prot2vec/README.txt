prot2vec.py takes input tab-delimited text files and performs calculates a word2vec embedding of a specificied N-gram size to a specified output vector size. 

Columns of the .tsv files 1. Organism 2. Amino Acid Sequence, over all of the sequences in the .fasta files. 

uniprot_sprot.fasta.gz and uniprot_sprot.tsv.gz are compressed files containing all of the sequences for all possible organisms from the reviewed (i.e. Swiss-Prot) sequences in UniProtKB (downloaded from https://www.uniprot.org/help/downloads)
